# 🚦 ESP32 Smart Toll Gate Automation

## Overview

An IoT-based Toll Gate Automation System built using ESP32, HC-SR04 Ultrasonic Sensor, IR Sensor, RGB LED, and SSD1306 OLED Display.

## Features

* Vehicle detection using ultrasonic sensor
* IR obstacle sensing
* OLED status display
* RGB LED state indication
* FreeRTOS task-based implementation
* ESP32-based control system

## Hardware Components

* ESP32 Development Board
* HC-SR04 Ultrasonic Sensor
* IR Obstacle Sensor
* SSD1306 OLED Display
* RGB LED
* Jumper Wires

## Project Files

* `Toll_get` – Main ESP32 source code
* `day-1` – Initial project version
* `TollGate_PinDatabase.docx` – Pin mapping documentation
